# project1_Eroshkin_Kirill_M25-555
## Лабиринт сокровищ
Игрок блуждает по извилистому лабиринту, собирает предметы, разгадывает хитроумные загадки, избегает коварных ловушек и в итоге стремится открыть таинственный сундук с сокровищами.
## Установка

```bash
poetry install
# или
make install
```
